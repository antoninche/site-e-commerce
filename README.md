# NIKE (demo) — Site e-commerce (HTML/CSS/JS)

⚠️ **Projet démo** (portfolio / apprentissage).
Style **inspiré** (minimal noir/blanc), sans logo officiel.

## Fonctionnalités

- Catalogue produits (recherche, tri, filtres multicritères).
- Panier persistant via `localStorage`.
- Favoris persistants avec compteur global.
- Filtre prix maximum + filtre favoris uniquement.
- Cartes enrichies (notes/avis + quantité rapide).
- Page panier avec codes promo démo (`WELCOME10`, `SHIPFREE`).
- Accessibilité améliorée (skip-link, labels, états ARIA).
- Préférence utilisateur "réduction des animations".

## Lancer le projet

Ce projet est full statique.

1. Ouvrir `sources/index.html` directement dans le navigateur.
2. Ou lancer un serveur statique local:

```bash
python -m http.server 8000 --directory sources
```

Puis ouvrir `http://localhost:8000`.
