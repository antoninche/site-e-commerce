# Site e-commerce (front)

Prototype de site e-commerce en HTML/CSS (et éventuellement JavaScript) : page produit / liste / panier (selon ton implémentation).

## Objectifs
- Construire une UI claire (cards produit, navigation, panier)
- Travailler le responsive et l’ergonomie
- (optionnel) Simuler un panier côté front

## Technologies
- HTML5
- CSS3
- (optionnel) JavaScript (DOM)
